# Collegeapp

A college admissions website where students can save and work on their applications, powered by the Gemini API.

## Status

🚧 Initial scaffold. The full application is built from a project prompt (in progress).

## Deployment

This repo auto-deploys to **Netlify** on every push to `main`.

> **Security note:** The Gemini API key must **never** be committed to this public repo or shipped in client-side code. It will live as a Netlify environment variable and be used only from a serverless function (Netlify Function) so it stays server-side.
